# Study Guide 1: Ethics

This module is about the responsible use of AI coding assistants. Instead of
teaching a new C++ or Python concept, it asks you to experience firsthand what
happens when you ship code you do not understand -- so-called "vibe coding" --
so that you build a healthy skepticism toward code-generation tools before you
rely on them in later, harder coursework. By design, this module requires no
programming knowledge at all: you are meant to lean entirely on an AI tool (or
a friend) and then reflect on what that felt like and what it did not teach you.

## Know before you start

Nothing. That is the point: the assignment is deliberately sequenced before
any C++ is taught, and you are expected to complete it without understanding
the code you submit.

## Taught here

Concept: AI-assisted coding and its risks
- Know that "vibe coding" means generating code with an AI assistant and
  shipping it without fully understanding how it works.
- Know that AI-generated code can compile and look plausible while still
  containing real bugs -- you have no way to judge which without the
  knowledge the tool is standing in for.
- Know that relying on code generation in a course designed to teach you
  to program creates a skill gap that compounds in later coursework and
  in professional work.
- Know that when you eventually understand the material (this same broken
  class is fixable by hand after the Dynamic Memory module), you can look
  back at what the AI did and judge it -- until then, you are trusting
  output you cannot evaluate.

Concept: honest use of tools
- Know your course and institution rules for when AI assistance is
  permitted, required to be disclosed, or prohibited.
- Be able to articulate the difference between using a tool to LEARN
  (explanations, examples you then rewrite) and using it to AVOID
  learning (submitting output unread).

## Study checklist

- [ ] Explain in your own words why shipping AI-generated code you do not
      understand is risky in a course designed to teach you to program.
- [ ] State when AI assistance must be disclosed under your course rules.
- [ ] After finishing: name one thing the AI-generated fix did that you
      could not evaluate at the time.
