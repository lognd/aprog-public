# Add Two Numbers

Read two integers from standard input and print their sum.

## Problem Statement

Your program will receive two integers, one per line. Print their sum followed
by a newline.

## Input Format

- Line 1: an integer `a`
- Line 2: an integer `b`

## Output Format

Print `a + b` on a single line.

## Examples

### Example 1

**Input:**
```
3
7
```

**Output:**
```
10
```

### Example 2

**Input:**
```
-5
3
```

**Output:**
```
-2
```

## Constraints

- `-10^9 <= a, b <= 10^9`

## Grading

| Component   | Points |
|-------------|--------|
| Correctness | 100    |

## Submission

Submit a single file named `add-two-numbers.py`.
