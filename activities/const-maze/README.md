# Activity: Const Maze

## What you will practice

The `const` keyword in C++ and how its position in a declaration changes its
meaning. You will also distinguish `const` from `constexpr` and understand why
`const` references are the preferred way to pass large objects to functions.

## How it works

Eight questions on const pointer declarations, const member functions, const
references, and constexpr. Each question has a hint if you are unsure where
to start. The activity unlocks when all answers are correct.

## Running the activity

```
python3 launch.py
```

No special tools required.
